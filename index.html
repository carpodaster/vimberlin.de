»Next meetup is something special:
"VIM NOVICE - a beginners workshop for people new to Vim"
2012-08-29, 7 p.m. at LAUNCH/CO (map)«

After discussing with @lucapette and @guckes on our last meetup, we brainstormed the ideas how we can teach Vim not only to basic users of the editor but as well to beginners who has never used a modal editor like Vim before.

We will have three coaches:

  • @lucapette - more than over 5 years of Vim experience, has written a bunch plugins for Vim, and using editor heavenly on hacking on different Rails projects.
  • @guckes - countless years of experience in using Vim and he can convince you in two minutes to use another shell then bash.
  • @wikimatze - using Vim for two years now and using it to write a book about Padrino.

Schedule:

  • Get to know each other - tell us something about yourself and what editors you have used in the past.
  • Tell what you want to learn (or what you already know) and we will start.
  • Find more about the topic under our wiki.

If you are interested, please sign under evenbrite and lanyrd we have place for 12 students. It would be nice if you already installed Vim on your operation system so that we can right start without any bigger technical problems. You can find the installation instruction for Vim here.

If you have any questions, please contact matthias.guenther@wikimatze.de.

You can find and track this event on lanyrd and you can register there with your twitter account.

Thanks to LAUNCH/CO for hosting the event.

Matthias
